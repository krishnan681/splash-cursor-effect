# ✨ SplashCursor using React + Vite

A beautiful interactive **fluid splash cursor effect** built with **React + Vite** using pure CSS.

## ✨ Features

- 🎯 Real-time fluid splash cursor
- ⚡ High-performance canvas rendering
- 🎨 Pure CSS styling (no Tailwind)
- 📱 Fully responsive
- 🖱️ Smooth hover interactions


## 🚀 Getting Started

### 1. Clone the repo
git clone <your-repo-url>
cd splashcursor

### 2. Install dependencies
npm install

### 3. Run locally
npm run dev
